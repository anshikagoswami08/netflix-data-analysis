# Netflix Data Analysis

## Overview
Analyzed Netflix dataset to identify content trends, genre distribution, and growth patterns.

## Tools Used
- SQL
- Python (Pandas)

## Dataset
- 8,000+ Netflix titles
- Includes genre, country, release year

## Key Insights
- Drama and International content dominate Netflix library
- Content production increased significantly after 2015
- US and India are top content producers

## Files Included
- CSV dataset
- SQL queries
- Python script
